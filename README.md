# Calculo de IMC Massa corpória 

# Sobre o projeto 

>IMC.PHP e um projeto realizado para calculo de massa corpória 


>Status: Developing ⚠️


# Como executar o projeto

```

<?php

$peso = 60;
$altura = 1.74;
$imc = $peso / $altura ** 2;

echo "Seu IMC é de $imc. Você está com o IMC ";

if ($imc < 18.5) {
    echo "abaixo";
} elseif ($imc < 25) {
    echo "dentro";
} else {
    echo "acima";
}

echo " do recomendado";

```
